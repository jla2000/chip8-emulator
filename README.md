# chip8-emulator


My own take on a very simple chip8 emulator, using rust and the wgpu project.


## Features

- Cross-platform
- Fast wgpu backend
- Working Keyboard & Audio


## Usage

Download any chip8 rom from the internet.
Eg. _space_invaders.ch8_:

```sh
cd chip8-emulator
cargo run -- space_invaders.ch8
```
